# Part 5 / Clip 1: CLAUDE.md — AI에게 업무 매뉴얼 쓰기

> 강의 영상: Part 5 / Ch 01 / Clip 1 (~20분)
> 만드는 것: 본인 학습 워크스페이스용 CLAUDE.md (`~/fastcampus-cc/50-my-work/Part05-뜯어보기/my-cc-study/CLAUDE.md`)

## 🎯 이 클립에서 만드는 것

지금까지 강의 워크스페이스에서 실습하셨죠. 한번 돌아보면 신기한 게 많아요. 클립 끝내고 `완료`만 입력하면 다음 클립으로 자동 안내되고, 진도가 자동으로 체크됩니다. 새 실습 시작하면 결과물 폴더가 알아서 만들어지고, 필요한 파일도 자동으로 준비돼 있었어요. AI는 항상 한국어로 답했고, 결과물은 정해진 위치에 알아서 저장됐죠.

이게 다 **한 파일에 적힌 약속** 덕분입니다. 이름은 `CLAUDE.md`. 워크스페이스 루트에 있는 마크다운 파일 한 장이에요. 클로드코드가 새 세션을 열 때마다 이 파일을 자동으로 읽고, 거기에 적힌 규칙대로 동작합니다.

오늘 만드는 건 **본인 학습 워크스페이스용 CLAUDE.md**예요. 강의 워크스페이스에 있는 건 강의 진행용이라 손대지 않습니다. 대신 별도로 본인 공부용 워크스페이스를 받아서, 거기에 본인 CLAUDE.md를 작성해요.

| Before | After |
|---|---|
| 새 세션마다 본인 직업, 답변 스타일을 매번 다시 알려줘야 함 | 한 번 적어두면 다음 세션부터 자동 적용 |
| 강의 워크스페이스가 왜 알아서 한국어로 답했는지 모름 | CLAUDE.md 안 3섹션(기본 규칙 / 자연어 트리거 / 결과물 저장) 직접 확인 |
| 강의 끝나면 어디서 클로드코드 공부해야 할지 막연 | `my-cc-study/` 폴더가 강의 후 평생 쓸 본인 공부방 |

이번 클립 핵심은 두 가지예요. **CLAUDE.md가 어떻게 동작하는지 강의 워크스페이스의 실물로 확인**하고, **본인 학습 워크스페이스에 직접 한 장 작성**하는 것.

## 💡 핵심 개념

### CLAUDE.md는 워크스페이스 약속 노트입니다

한 줄로 정리하면 "이 폴더에서 일할 땐 이렇게 해"라는 매뉴얼이에요. 클로드코드가 새 세션을 시작할 때마다 자동으로 읽습니다. 본인이 매번 입력으로 알려줄 필요가 없어요.

위치는 두 곳에 둘 수 있어요.

| 위치 | 적용 범위 |
|---|---|
| 워크스페이스 루트 (`./CLAUDE.md`) | 그 폴더에서 작업할 때만 |
| 사용자 홈 (`~/.claude/CLAUDE.md`) | 모든 워크스페이스 (전역) |

오늘은 워크스페이스 루트 쪽을 만듭니다. `my-cc-study/CLAUDE.md`. 이 파일이 있으면 `my-cc-study` 안에서 클로드코드를 켤 때마다 자동으로 적용돼요.

### 강의 워크스페이스 CLAUDE.md 3섹션

본인 거 만들기 전에, 잘 만들어진 예시를 한 번 보고 갑니다. 강의 워크스페이스 루트에 있는 CLAUDE.md를 안티그래비티 에디터에서 열어보세요. **읽기만 합니다. 절대 수정하지 마세요.** 이 파일을 건드리면 강의 진행이 깨져요.

핵심 섹션 세 개만 짚고 갈게요.

**섹션 1. 기본 규칙**

```text
- 모든 답변은 한글로 합니다
- 전문 용어는 쉬운 말로 풀어서 설명합니다
- 결과물은 눈에 보이는 형태로 만듭니다
```

이 세 줄 때문에 강의 내내 한국어로 답했고, 시각적 결과물이 나온 거예요. 별도 명령 없이도 매 세션마다 적용됐던 거고요.

**섹션 2. 자연어 트리거**

```text
| `막혔어요` | `/stuck` | 에러/문제 진단 |
```

"막혔어요" 한 마디 치면 자동으로 `/stuck` 슬래시 명령이 실행됐던 거 기억나시죠? 그것도 이 줄 한 줄 덕분이에요. 자연어 표현을 슬래시 명령에 매핑해두는 패턴입니다.

**섹션 3. 결과물 저장 규칙**

```text
모든 실습 결과는 50-my-work/Part{NN}-제목/실습{NN}-제목/에 저장합니다.
```

AI가 알아서 폴더 만들고 저장한 게 이 규칙 한 줄 때문이에요. 본인이 매번 "어디에 저장해줘"라고 안 적어도 됐던 이유.

이 세 섹션이 CLAUDE.md의 기본 골격이에요. **기본 규칙 + 자연어 트리거 + 저장 규칙**. 본인 거 만들 때도 이 골격을 따라가면 됩니다.

### 두 워크스페이스 역할 분리

여기서 한 번 짚고 갈 게 있어요. 이번 Part 5부터는 워크스페이스가 두 개입니다.

| 워크스페이스 | 경로 | 용도 | 수정 |
|---|---|---|---|
| 강의 워크스페이스 | `~/fastcampus-cc/` | 강의 진행용 | ❌ 절대 X |
| 학습 워크스페이스 | `~/fastcampus-cc/50-my-work/Part05-뜯어보기/my-cc-study/` | 본인 공부용 | ✅ 자유 |

강의 워크스페이스의 CLAUDE.md나 스킬 파일을 건드리면 강의가 깨져요. 본인 자산은 학습 워크스페이스에 자유롭게 만드시면 됩니다. 강의 끝나도 `my-cc-study/`는 본인 공부방으로 계속 써요.

## 🚶 진행 흐름

### 1. `/part05` 스킬 호출 → 학습 워크스페이스 자동 준비

안티그래비티에 강의 워크스페이스가 켜져 있고, 클로드코드도 띄워둔 상태에서 시작합니다.

```text
/part05
```

스킬이 진도를 확인하고 클립 선택지를 띄워요. **Clip 1**을 선택합니다.

스킬이 알아서 학습 워크스페이스를 만들어줘요. `cp -r` 같은 명령을 직접 칠 필요 없습니다.

```text
✓ ~/fastcampus-cc/50-my-work/Part05-뜯어보기/my-cc-study/ 시드 복사 완료
✓ 50-my-work/Part05-뜯어보기/실습18-CLAUDEmd/ 진도 폴더 준비 완료
README.md  10-curriculum/  20-concepts/  30-resources/  40-ideas/  sandbox/
```

여기서 한 가지 짚을 게 있어요. **`CLAUDE.md`만 일부러 빠져 있습니다**. 우리가 직접 만들 차례라서 그래요.

### 2. 강의 워크스페이스 CLAUDE.md 같이 보기

본격 작업 전에 잘 만들어진 예시를 봅니다. 안티그래비티 에디터에서 강의 워크스페이스 루트의 `CLAUDE.md`를 여세요. 위에서 본 3섹션(기본 규칙 / 자연어 트리거 / 결과물 저장)이 어떻게 적혀 있는지 직접 확인합니다.

다시 강조해요. **읽기만 하고 닫습니다.** 수정 X.

### 3. 안티그래비티 새 창에서 학습 워크스페이스 열기

이제 학습 워크스페이스로 갑니다. 안티그래비티에서 **새 창**을 하나 열고, **Open Folder**로 `~/fastcampus-cc/50-my-work/Part05-뜯어보기/my-cc-study/`를 선택해요.

새 창이 떴으면 안티그래비티 통합 터미널에서 확인합니다.

```bash
ls
```

`README.md`, `10-curriculum/`, `20-concepts/`, `30-resources/`, `40-ideas/`, `sandbox/`가 보이는데 **`CLAUDE.md`만 없죠?** 좋아요. 이게 정상이에요.

### 4. `/init`으로 CLAUDE.md 초안 만들기

학습 워크스페이스 안에서 클로드코드를 실행합니다.

```bash
claude
```

여기는 강의 워크스페이스와 분리된 새 세션이에요. 클로드코드가 시작되면 `/init`을 입력합니다.

```text
/init
```

클로드코드가 1분 정도 폴더를 분석합니다. "학습 노트 / 자료 / 아이디어 정리 워크스페이스로 보입니다" 같은 진단을 내고, CLAUDE.md 초안을 제안해요. 그대로 저장하면 됩니다.

안티그래비티 에디터로 돌아와 새로 생긴 `CLAUDE.md`를 열어보세요. 폴더 구조 기반의 초안이 들어 있을 거예요. 다만 이건 폴더만 보고 추측한 초안이라, 본인 맥락이 더 반영돼야 진짜 본인 워크스페이스가 됩니다.

### 5. 인터뷰로 본인 맥락 반영

학습 워크스페이스의 클로드코드 세션에서 인터뷰를 요청합니다.

```text
내가 클로드코드 공부하는 사람이라는 맥락이 더 반영되도록
CLAUDE.md를 인터뷰로 보완해줘.

한 번에 하나씩 5개 질문 후 종료.
- 내 직업/배경
- 답변 스타일 선호 (쉬운 말? 예시 많이?)
- 결과물 포맷 선호 (표/리스트/문서)
- 자주 다룰 주제
- 피해야 할 것
```

다섯 개 질문이 한 번에 하나씩 옵니다. 답변 가이드는 이렇게 잡아주세요.

| Q | 답변 가이드 |
|---|---|
| Q1. 직업/배경 | 본인 직업 한 줄. 가상 프로필도 OK (예: 마케팅 기획자, 콘텐츠 운영자, 직장인) |
| Q2. 답변 스타일 | 존댓말 / 쉬운 말 / 예시 많이 등 본인 선호 |
| Q3. 결과물 포맷 | 표 / 리스트 / 문서 / 3줄 요약 등 |
| Q4. 자주 다룰 주제 | 본인 업무·관심 분야 |
| Q5. 피해야 할 것 | 너무 긴 답변 / 영어 전문 용어 그대로 / 모호한 설명 등 |

다섯 개 다 답하면 AI가 CLAUDE.md를 다듬어서 저장합니다. 안티그래비티에서 다시 열어 변경된 내용을 확인하세요.

### 6. Before/After — 진짜 동작하는지 확인

마지막으로 효과 체감입니다. 클로드코드를 한 번 재시작해서 새 세션에서 같은 질문을 던져봐요.

```bash
exit
claude
```

새 세션이 시작되면 이렇게 물어봅니다.

```text
클로드코드를 더 공부하려면 뭐부터 보면 좋을까?
```

본인이 인터뷰에서 답한 직업, 답변 스타일, 자주 다룰 주제가 반영된 답이 나올 거예요. 같은 질문인데 본인 맥락 따라 답이 달라지는 게 보이실 거고요. **CLAUDE.md의 힘이 이거예요. 같은 질문, 다른 대화.**

## 🆘 자주 막히는 포인트

| 증상 | 원인 | 해결 |
|---|---|---|
| `/part05` 호출했는데 안 먹음 | 강의 워크스페이스 밖에서 호출 | 안티그래비티에서 강의 워크스페이스 폴더 열고 클로드코드 실행 |
| 학습 워크스페이스 이미 있음 | 이전 시도 흔적 | 스킬이 자동으로 `.bak.YYYYMMDD-HHMMSS`로 백업하니 신경 안 써도 됨 |
| `/init` 결과가 영어로 나옴 | 클로드코드 기본값 | "한국어로 다시 써줘" 입력 |
| `/init` 자체가 안 먹음 | 클로드코드 버전 낮음 | `claude --version`으로 확인하고 업데이트 |
| 인터뷰가 2~3개 질문만 하고 끝남 | AI 해석 차이 | "5개 다 채워줘" 한 번 더 요청 |
| Before/After 차이가 안 느껴짐 | CLAUDE.md가 너무 두루뭉술 | "더 구체적으로 다듬어줘" 추가 요청 |
| 인터뷰 답변에 개인정보 걱정 | 프라이버시 | 가상 프로필로 답해도 됨. "민감 정보 빼고" 명시 |
| 안티그래비티 새 창을 어떻게 여는지 | IDE 사용 미숙 | File > New Window 또는 단축키. 폴더 열기는 File > Open Folder |
| 강의 워크스페이스 CLAUDE.md를 수정해버림 | "내 거" 혼동 | git에서 복원. 본인 자산은 학습 워크스페이스에 만듦 |
