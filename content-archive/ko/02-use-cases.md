# 02. Claude Code로 할 수 있는 것들

> "Code"라는 이름과 달리, Claude Code는 코딩 전용 도구가 아닙니다. 터미널 위에서 돌아가는 <strong>범용 AI 에이전트</strong>입니다.

---

## 먼저: 웹 UI(Claude.ai)와 뭐가 다른가요?

| | Claude.ai (웹) | Claude Code (터미널) |
|--|---------------|---------------------|
| 파일 접근 | 직접 첨부만 가능 | 폴더 전체 읽기·수정 |
| 작업 범위 | 대화 1회 | 수십 개 파일 동시 처리 |
| 외부 실행 | 불가 | 터미널 명령어 직접 실행 |
| 반복 자동화 | 매번 수동 | Skills·Hooks으로 자동화 |
| 기억 | 대화 안에서만 | CLAUDE.md로 영구 기억 |

한마디로: <strong>Claude.ai가 대화 상대라면, Claude Code는 내 컴퓨터에서 일하는 AI 동료</strong>입니다.

---

## 개발 환경에서

### 코드 작성 & 관리
코드 작성, 버그 수정, 리팩토링, 테스트, 코드 리뷰 — 개발자에게 익숙한 모든 작업을 자연어로 지시할 수 있습니다.

### 바이브코딩 — 코드를 몰라도 소프트웨어를 만들 수 있습니다

"분위기(vibe)로 코딩한다"는 뜻. <strong>코드를 직접 작성하지 않고 AI에게 자연어로 지시해서 소프트웨어를 만드는 방식</strong>입니다. Andrej Karpathy(전 Tesla AI 총괄)가 2025년에 이름 붙인 개념입니다.

| | 개발자 | 바이브코더 |
|--|--------|-----------|
| 코드 작성 | 직접 작성 | AI에게 지시 |
| 디버깅 | 에러 메시지 분석 → 수정 | "이거 에러 나는데 고쳐줘" |
| 핵심 역할 | 구현 | **기획 & 검수** |
| 필요한 능력 | 프로그래밍 언어 | **명확한 요구사항 전달** |

바이브코딩에서 가장 중요한 건 코딩 실력이 아니라 <strong>"뭘 만들고 싶은지 명확하게 설명하는 능력"</strong>입니다.

**진짜 할 수 있는 것**: 프로토타입, MVP, 개인용 도구, 웹사이트, 간단한 앱, 데이터 처리 자동화. **이 사이트(CC101)도 바이브코딩으로 만들었습니다.**

**솔직한 한계**: 대규모 상용 서비스는 전문 개발자 검수가 필요합니다. 보안, 결제 등 민감한 로직은 반드시 전문가 리뷰를 받으세요. 바이브코딩으로 시작하되, 서비스가 커지면 개발자와 협업하는 게 현명합니다.

### 바이브코딩 잘하는 법

1. **기획이 반이다** — "앱 만들어줘"보다 "사진을 날짜별로 정리하는 웹앱 만들어줘"가 훨씬 좋은 결과를 냅니다. show-me-the-prd 플러그인으로 기획서를 자동 생성할 수 있습니다.
2. **작게 시작해서 키워라** — 한 번에 완성품을 요구하지 마세요. "로그인 화면 먼저" → "다음은 목록 화면" 순서로 점진적으로.
3. **결과를 직접 확인하라** — AI가 만든 결과물을 반드시 실행하고 확인하세요.
4. **실수를 두려워하지 마라** — Git으로 저장해두면 언제든 되돌릴 수 있습니다.

> 바이브코딩 활용법에 대해 궁금하다면 [클로드코드 뽀개기 (패스트캠퍼스)](https://fastcampus.co.kr/biz_online_claudecode)를 참고하세요.

## 비개발 환경에서

**Claude Code는 바이브코딩 이상의 것을 할 수 있습니다.** 코딩과 상관없이, 컴퓨터로 할 수 있는 거의 모든 작업을 자연어로 시킬 수 있습니다.

### 리서치 & 분석
- 여러 웹페이지를 동시에 읽고 마크다운으로 정리
- 경쟁사 분석, 시장 조사, 기술 동향 파악
- PDF·문서 수십 개 일괄 요약

> 💡 **deep-research** 플러그인을 사용하면 멀티에이전트가 여러 소스를 병렬로 조사해 종합 보고서를 자동 생성합니다.

### 문서 & 콘텐츠 제작
- 블로그 포스트, 뉴스레터, SNS 콘텐츠 초안
- 제안서, 보고서, 발표 자료
- 한국어 ↔ 영어 번역 후 파일로 자동 저장

### 데이터 처리
- CSV, Excel 분석 및 변환 — **파이썬 몰라도 됩니다**
- 파일 수백 개 일괄 처리 (이름 변경, 형식 변환, 폴더 정리)
- JSON, XML 데이터 파싱 및 가공

### 학습 보조
- 강의 자료, 책 내용 구조화 정리
- 퀴즈, 플래시카드 자동 생성
- 개념 설명 + 예시 + 연습 문제

### 비즈니스 워크플로우
- 미팅 녹취 텍스트 → 요약 + 결정사항 + 담당자별 액션 아이템
- 주간·월간 보고서 자동화
- 이메일 초안, 고객 응대 템플릿

### 개인 AI 워크스페이스
CLAUDE.md로 역할과 규칙을 정의하면, 단순 AI 채팅이 아닌 <strong>나만의 AI 에이전트 시스템</strong>이 됩니다.
→ 나만의 AI 워크스페이스 설계 섹션에서 자세히 다룹니다

## 실제 사례: CC101은 어떻게 만들어졌나

> **코드를 한 줄도 직접 짜지 않았습니다**

**기획**
"Codex 101 같은 한국어 Claude Code 가이드를 만들고 싶다"
→ 아이디에이션 워크스페이스에서 3개 AI(Claude·Codex·Gemini)가 섹션 구성, UX, 기술 스택을 토론
→ 23개 섹션 구조 결정

**콘텐츠 제작**
공식 문서 57개 수집 → Claude가 핵심 내용 추출 → 한국어·영어 마크다운으로 구조화

**개발**
Next.js 사이트 전체 코드 → Claude Code가 작성
다크/라이트 모드, 한영 토글, 모바일 대응 → 모두 대화로 처리

**배포**
Vercel 배포, Cloudflare DNS 설정, 도메인 연결 → Claude Code가 직접 처리

**운영 & 보완**
초안 이후 사용자가 내용을 검토하고 "이 부분 추가해줘", "이건 다르게 써줘"로
대화를 통해 지속적으로 개선 중 — **지금 이 대화도 CC101 업데이트 세션입니다**

## 비개발자 실사용 예시

```
상황: 매주 팀 미팅 후 회의록을 정리해야 한다

기존 방식:
  미팅 끝 → 메모 정리 30분 → 이메일 작성 10분 → 공유

Claude Code 방식:
  미팅 녹취 텍스트 붙여넣기
  → "요약, 결정사항, 담당자별 액션 아이템 정리해줘"
  → 마크다운 파일로 저장 + 이메일 초안 자동 생성
  총 소요: 3분
```
