# 알아두면 유용한 기능들

> Claude Code를 더 편하게 쓸 수 있는 기능들을 모았습니다.

---

## 입력을 편하게

### 이미지 입력

에러 스크린샷, UI 디자인 시안 등을 Claude에게 직접 보여줄 수 있습니다.

- **Ctrl+V** — 클립보드 이미지 붙여넣기 (macOS에서도 Cmd+V가 아닌 Ctrl+V)
- **드래그 앤 드롭** — 이미지 파일을 터미널 창으로 끌어다 놓기
- **파일 경로** — `"이 이미지 분석해줘: /path/to/image.png"`
- 지원 형식: JPEG, PNG, GIF, WebP (최대 5MB)

```
> 이 스크린샷에서 에러 원인 찾아줘 [Ctrl+V로 붙여넣기]
> @design-mockup.png 이 시안대로 만들어줘
```

### 음성 입력 (/voice)

타이핑 대신 말로 지시할 수 있습니다.

- `/voice` 입력 후 **스페이스바를 누른 채** 말하기
- 한국어 포함 20개 언어 지원
- 음성과 타이핑 혼합 사용 가능
- Beta 기능 (점진적 확대 중)

> 긴 설명을 타이핑하기 귀찮을 때 특히 유용합니다.

## 실수 복구

### 체크포인트 (Esc Esc)

Claude가 파일을 수정하기 전에 자동으로 스냅샷을 저장합니다.

- **Esc 키를 두 번** 누르면 직전 상태로 되돌리기
- 별도 설정 없이 자동으로 동작
- git을 몰라도 되돌릴 수 있는 가장 빠른 방법

### /rewind

대화와 코드를 이전 시점으로 되돌립니다.

- 여러 단계를 거슬러 올라가야 할 때 유용
- `Esc Esc`가 직전 한 단계 되돌리기라면, `/rewind`는 원하는 시점까지 되돌리기

```
> /rewind
→ 시점 목록이 나타나면 원하는 곳을 선택
```

## 세션이 길어질 때

### 자동 압축 (auto-compact)

대화가 길어지면 Claude가 자동으로 오래된 내용을 정리합니다.

- 별도 설정 없이 자동으로 동작
- CLAUDE.md에 적어둔 규칙은 삭제되지 않고 보존
- 수동으로 하려면 방향을 지정할 수 있습니다:

```
> /compact API 변경 중심으로 정리해줘
```

### 자동 메모리 (Auto Memory)

Claude가 작업 중 중요한 정보를 자동으로 기억합니다.

- CLAUDE.md와 별개로 자동 동작
- 세션이 바뀌어도 이전 작업 맥락을 기억
- `/memory` 명령으로 저장된 메모리 확인/편집 가능

### Extended Thinking (/effort)

복잡한 문제에서 Claude가 더 깊이 생각하도록 설정합니다.

- 기본적으로 활성화되어 있음 (별도 설정 불필요)
- `/effort high` — 어려운 문제에 깊이 생각 (느리지만 정확)
- `/effort low` — 간단한 질문에 빠르게 응답
- 프롬프트에 "ultrathink"를 넣으면 해당 턴만 최고 깊이로 사고

## 안전하게 쓰기

### 권한 모드 (Permission Modes)

Claude가 파일을 수정하거나 명령을 실행하기 전에 확인을 받습니다.

| 모드 | 설명 |
|------|------|
| **Ask (기본값)** | 매번 확인 — 초보자에게 안전 |
| **Auto-accept** | 신뢰할 수 있는 작업은 자동 승인 |
| **Plan** | 실행 전에 계획을 먼저 보여주고 승인 |

- `/permissions`로 모드 전환 가능
- 처음에는 기본값(Ask)으로 사용하면서 익숙해지세요

## 이런 것도 됩니다

**Computer Use** — Claude가 화면을 보고 마우스/키보드를 직접 조작합니다. 앱 테스트, GUI 자동화에 유용합니다. (Research Preview, macOS)

**Remote Control** — 터미널에서 시작한 작업을 폰이나 다른 기기에서 이어할 수 있습니다. QR 코드 스캔으로 연결합니다.

**Chrome 연동** — Claude가 Chrome 브라우저를 직접 제어합니다. 웹 테스트, 데이터 추출, 폼 자동 입력에 유용합니다.

**예약 작업 (/loop)** — 특정 프롬프트를 정해진 간격으로 반복 실행합니다. 모니터링, 정기 체크에 유용합니다.
