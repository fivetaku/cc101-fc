# 03. 요금제 안내

> Claude Code를 사용하려면 유료 구독이 필요합니다. 본격적으로 쓰려면 Max $100을 추천합니다.

---

## Pro vs Max 플랜 비교

> 최신 가격은 [claude.com/pricing](https://claude.com/pricing)에서 확인하세요. 아래는 참고용입니다.

| | **Pro** | **Max (5x)** | **Max (20x)** |
|---|---|---|---|
| **월 가격** | $20 | $100 | $200 |
| **사용량 한도** | 기본 | Pro의 5배 | Pro의 20배 |
| **Claude Code 포함** | 포함 | 포함 | 포함 |
| **Claude.ai 웹 포함** | 포함 | 포함 | 포함 |
| **적합한 사용자** | 체험, 가벼운 사용 | Claude Code 주 사용자 | 헤비유저, 대규모 프로젝트 |

### 입문자 추천: Max $100

Claude Code를 본격적으로 사용하려면 <strong>Max $100 플랜</strong>을 권장합니다.

- Pro($20)는 사용량 한도가 빠르게 차서 집중 작업이 어렵습니다.
- 가장 강력한 Opus 모델은 Pro에서 사실상 사용 불가능합니다.
- 멀티세션, 에이전트 팀 같은 생산성 기능은 토큰을 빠르게 소모합니다.
- Max $100는 대부분의 개인 사용자에게 충분합니다.

---

## 비용 구조

```
월정액 결제 → 정해진 사용량 한도 내에서 자유롭게 사용
한도 초과 시 → 추가 응답 제한 (추가 과금 없음)
```

- 토큰 수를 일일이 신경 쓸 필요 없습니다
- 한도에 도달하면 일정 시간 후 리셋됩니다
- `/cost` 명령어로 현재 세션 사용량을 확인할 수 있습니다

---

## 비용 절감 팁

| 방법 | 설명 |
|------|------|
| 작업 사이에 `/clear` 실행 | 이전 대화 컨텍스트를 지워 토큰 절약 |
| Sonnet 모델 우선 사용 | Opus보다 저렴하고 대부분 작업에 충분 |
| 명확한 요청 | "이 코드베이스 전체 개선해줘" 대신 특정 파일/함수 지정 |
| MCP 서버 최소화 | 사용하지 않는 연결은 비활성화 |
